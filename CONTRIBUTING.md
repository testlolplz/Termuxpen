# Contributing to TermuxPen

First off, thank you for considering contributing! 🎉

## Code of Conduct
This project adheres to a [Code of Conduct](CODE_OF_CONDUCT.md). By participating, you agree to uphold this code.

## How Can I Contribute?

### Reporting Bugs 🐛
1. Check if bug already exists in [Issues](https://github.com/testlolplz/Termuxpen/issues)
2. Use the bug report template
3. Include:
   - Termux version
   - Python version
   - Steps to reproduce
   - Expected vs actual behavior
   - Screenshots if applicable

### Suggesting Features 💡
1. Check existing [feature requests](https://github.com/testlolplz/Termuxpen/issues?q=is%3Aissue+label%3Aenhancement)
2. Use feature request template
3. Explain use case clearly

### Pull Requests 🔧
1. Fork the repo
2. Create feature branch: `git checkout -b feature/amazing-feature`
3. Follow coding style:
   - Use 4 spaces for indentation
   - Keep functions under 50 lines
   - Add docstrings
   - Include error handling
4. Test your changes
5. Commit with clear messages: `git commit -m 'Add: amazing feature'`
6. Push and create PR
